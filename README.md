# Task Management App

## Backend Features
- User authentication (JWT)
- Task CRUD operations
- MongoDB integration
- Express + TypeScript

## API Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/:id
- PATCH /api/tasks/:id/status
- DELETE /api/tasks/:id
